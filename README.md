# House-Price-Prediction-System

## Overview
This project implements a House Price Prediction System using the Linear Regression machine learning algorithm. The model predicts house prices based on various housing features from the California Housing Dataset.

## Features
- Data preprocessing
- Handling missing values
- Categorical data encoding using `get_dummies()`
- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- Scatter Plot
- Train-Test Split
- Feature Scaling
- Linear Regression Model
- House Price Prediction
- Model Evaluation using Mean Squared Error (MSE) and R² Score

## Technologies Used
- Python
- Visual Studio Code (VS Code)
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
California Housing Dataset

## Files
- `test.py` – Main Python program
- `housing.csv` – Dataset
- `correlation_heatmap.png` – Correlation Heatmap
- `price_scatter_plot.png` – Scatter Plot
